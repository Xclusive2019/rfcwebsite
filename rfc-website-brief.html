1. Executive Summary

This brief outlines the complete redesign and rebuild of the RFC Food Safety Consultants website. The project addresses the key challenges identified in the UX assessment and competitive research: unclear brand architecture, weak homepage messaging, lack of interactive tools, and no direct booking capability.

What We Built

A 5-page professional website with an editorial design aesthetic, featuring all four RFC divisions prominently on the homepage, an integrated calendar booking system, an AI-powered chat assistant, and an interactive audit readiness quiz. Built with React, TypeScript, and Tailwind CSS for performance and maintainability.

2. Design Approach

The design follows an editorial consulting-firm aesthetic inspired by top-tier management consultancies. The approach prioritises clarity, professionalism, and restraint over decorative effects.

2.1 Visual Identity

Unlike typical websites that rely on gradients, floating animations, and coloured backgrounds, this design uses:

  • White space as a design element — generous margins and breathing room between sections
  • Thin rule separators — 1px lines in light grey to divide content without visual noise
  • Sharp corners — no rounded cards or pill shapes; all edges are square for a precise, professional feel
  • Restrained colour usage — RFC green appears only for the logo, primary buttons, and key accents

4. Four-Division Structure

The most significant change from the previous website is the clear presentation of all four RFC divisions. Previously, visitors had to dig through menus to understand the full scope. Now, all four divisions appear immediately on the homepage hero.

01
RFC Consulting
FSSC 22000, HACCP, R638, BRCGS certification and audits. Full-service consulting from gap assessment to certification.
02
RFC Academy
SAATCA-accredited online food safety training. 6 courses from R638 PIC to Internal Auditor, R1,200-R1,600 each.
03
Comply Cloud
South Africa's first cloud-based food safety compliance platform. Paperless audits, dashboards, automated alerts.
04
Pest Control Solutions
HACCP-compliant integrated pest management for food industry. Sister company with paperless documentation.

Navigation Integration

Each division is accessible from the main navigation dropdown under "Services." External divisions (Academy and Pest Control) open in new tabs. Internal divisions (Consulting and Comply Cloud) scroll to their homepage sections.

5. Key Features

5.1 Calendar Booking System

A complete consultation booking flow that eliminates the back-and-forth of email scheduling.

How It Works

  1. User selects a date from 10 available weekdays (weekends excluded)
  2. User selects a time slot from 9 options (08:00-16:00)
  3. User fills in name, email, phone, company, service interest, and optional notes
  4. Confirmation screen displays the booked date and time

Design Details

  • Two-column layout: calendar on the left, "What to Expect" sidebar on the right
  • Selected date/time highlighted in RFC green
  • Sidebar explains the 30-minute call process, gap assessment, and next steps
  • Sticky sidebar stays visible while scrolling through the form
Note on Backend

The current booking system captures form data on the frontend. To go live, a backend email service (e.g., SendGrid, Formspree, or a custom API) should be connected to deliver booking notifications to info@rfcsa.co.za.

5.2 AI Chat Assistant

A floating chat widget in the bottom-right corner that provides instant guidance and drives conversions.

Capabilities

  • Welcome message with 4 quick-action buttons: Book consultation, Learn services, Check audit readiness, Speak to a human
  • Guided conversations — multi-step flows for each division with follow-up options
  • Navigation links — chat can direct users to the booking page, audit quiz, WhatsApp, or external sites
  • Free-form input — users can type questions and receive contextual responses
  • Booking shortcut — "Book" button in the chat header for one-click navigation

7. SEO & Content Strategy

The website structure supports the comprehensive SEO strategy developed in the research phase. Key SEO features built into the site:

On-Page SEO

  • Semantic HTML — proper heading hierarchy (H1-H6), section tags, and ARIA labels
  • Meta-ready — each page has a unique title and description placeholder
  • Fast load times — static deployment with optimised assets improves Core Web Vitals
  • Mobile responsive — all layouts adapt from mobile to desktop
